# RESTful API endpoints
| Verb        | Path           | Endpoint  |
| ------------- |-------------| -----|
| GET   | /             | index |
| GET   | /stops        | view all stops |
| GET   | /stops/:id    | view one stop |
| GET   | /routes       | view all routes |
| GET   | /routes/:id   | view one route |
